# Diwan-i-Naksh  
## Delhi Epigraphy Archive

Diwan-i-Naksh is an interactive digital archive of historical inscriptions, monuments, and scripts from Delhi. It combines spatial visualization, textual documentation, and AI-assisted interpretation to make epigraphic heritage accessible for researchers, students, and the public.

The project presents inscriptions not only as geographic points on a map but also as readable textual records with historical, architectural, and cultural analysis.

---

## Features

### Interactive Map
- Leaflet-based map interface
- Monument markers with script-based color coding
- Clickable markers showing inscriptions and images

### Monument Archive (Text Interface)
- Sidebar list of monuments
- Detailed textual information:
  - Translation
  - Script type
  - Historical context
  - Architectural analysis
  - Cultural stories
  - Design interpretation
- Search functionality

### AI Epigraphy Assistant
- Built-in AI assistant powered by Groq (LLaMA 3.3)
- Answers questions about:
  - Delhi monuments
  - Epigraphy
  - Scripts
  - Architecture
  - History

### Accessibility Features
- Text-based archive for non-map users
- Structured academic layout
- Searchable database
